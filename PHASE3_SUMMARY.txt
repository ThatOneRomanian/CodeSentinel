================================================================================
  QUICK START
================================================================================

Option 1: Automated Setup (Recommended)
$ ./setup-gui.sh                          # ~2 minutes

Option 2: Manual Setup
$ pip install fastapi uvicorn python-multipart
$ cd gui && npm install

Running the Application:

Terminal 1 (Backend):
$ python -m sentinel.api.fastapi_bridge

Terminal 2 (Frontend):
$ cd gui && npm run dev

Then open http://localhost:3000 in your browser

================================================================================
  API ENDPOINTS
================================================================================

GET    /health                    Health check
GET    /api/config                Configuration

POST   /api/scans                 Start new scan
GET    /api/scans/{id}            Get scan results
DELETE /api/scans/{id}            Cancel scan
WS     /api/scans/{id}/progress   Real-time progress

GET    /api/findings/{id}         Get finding details
PATCH  /api/findings/{id}         Update finding status
POST   /api/findings/{id}/notes   Add finding notes

GET    /api/projects              Project history
GET    /api/projects/{path}/scans Scans for project
DELETE /api/projects/{path}       Delete project

  SUPPORT

Common Issues:

Q: "ModuleNotFoundError: fastapi"
A: Run: pip install fastapi uvicorn python-multipart

Q: "npm ERR!"
A: Run: cd gui && npm install

Q: "CORS error"
A: Ensure backend (port 8000) and frontend (port 3000) are both running

Q: "Cannot connect to API"
A: Check: curl http://localhost:8000/health

Q: "Hot reload not working"
A: Restart: npm run dev (in gui/)
